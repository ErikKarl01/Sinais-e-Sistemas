# 📚 Sinais e Sistemas - Atividades e Resoluções

Bem-vindo ao repositório de atividades da disciplina de **Sinais e Sistemas**.

## 🎯 Sobre o Projeto
Este repositório foi criado para armazenar e organizar todas as resoluções de listas de exercícios, simulações numéricas e plotagens de gráficos desenvolvidos em grupo ao longo da matéria. O objetivo é manter um registro de aprendizado combinando a teoria matemática (resoluções analíticas) com a prática computacional (métodos numéricos).

## 🗂️ Estrutura do Repositório
O conteúdo está organizado de acordo com as unidades da disciplina:

* **`/Unidade_1`**
* **`/Unidade_2`**
* **`/Unidade_3`**

Dentro de cada diretório de unidade, você encontrará:
* `resolucoes_manuais/`: PDFs ou imagens com o desenvolvimento matemático das questões (método clássico, limites de integração, etc.).
* `notebooks/`: Arquivos Jupyter Notebook (`.ipynb`) com as simulações, computação simbólica e geração de gráficos solicitados nas atividades.

## 🛠️ Tecnologias e Bibliotecas Utilizadas
* **Linguagem:** Python 3.x
* **Ambiente:** Jupyter Notebook
* **Bibliotecas Principais:**
  * `numpy`: Trabalhar com vetores, tempo discretizado e convolução numérica.
  * `matplotlib`: Plotar os sinais no domínio do tempo e frequência.
  * `sympy`: Resolver equações diferenciais e integrais através de computação simbólica.

## 🚀 Como Executar Localmente
Se você quiser rodar os notebooks na sua máquina para interagir com os códigos e ver os gráficos, siga os passos abaixo:

1. Clone este repositório:
   ```bash
   git clone [https://github.com/ErikKarl01/Sinais-e-Sistemas.git](https://github.com/ErikKarl01/Sinais-e-Sistemas.git)
2. Instale as bibliotecas necessárias (recomenda-se o uso de um ambiente virtual):
   pip install numpy matplotlib sympy jupyter
3. Inicie o Jupyter Notebook:
   jupyter notebook
